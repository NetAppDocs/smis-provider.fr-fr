= crsp smis
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
Le `smis crsp` La commande décrit les sous-profils CIM-Registered pris en charge par le fournisseur NetApp SMI-S, y compris les sous-profils NetApp SMI-S Provider.

crsp smis

`[-t {*http* | *https*}]`

`C:\Program Files (x86)\NetApp\smis\pegasus\bin`

Utilisateur avec un nom d'utilisateur et un mot de passe valides

.Paramètres
* `*[-t {http | https}]*`
+
Protocole à utiliser : HTTPS (par défaut) ou HTTP
