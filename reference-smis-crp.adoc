= smis crp
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
Le `smis crp` La commande décrit les profils CIM-registered pris en charge par NetApp SMI-S Provider, y compris les profils NetApp SMI-S Provider.

smis crp

`[-t {*http* | *https*}]`

`C:\Program Files (x86)\NetApp\smis\pegasus\bin`

Utilisateur avec un nom d'utilisateur et un mot de passe valides

.Paramètres
* `*[-t {http | https}]*`
+
Protocole à utiliser : HTTPS (par défaut) ou HTTP



Le `smis crp` la commande et sa sortie :
